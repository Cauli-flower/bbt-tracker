🍽️ 稳血糖饮食卡

不节食掉秤 · 稳血糖 · 每餐够蛋白

⭐每餐三件套（最该记住的）

1先吃菜和肉，最后吃主食——同一碗饭，换个顺序，餐后血糖峰值就低一截。
2每餐蛋白质够一个手掌——蛋/肉/鱼/豆腐/酸奶，扛饿、稳血糖、长肌肉。
3饭后散步 10–15 分钟（尤其晚饭后）——肌肉把血糖用掉，最实用一招。

🥗三餐这样吃（白天足 · 晚上轻）

早餐 9:30
1个全蛋 + 蛋清 + 无糖豆浆/低脂奶 + 全麦夹奶酪（奶酪别多）。空腹练完，这顿把蛋白堆够。
午餐 15:00（别太轻）
无糖希腊酸奶 + 燕麦 + 磨碎亚麻籽 + 蓝莓 + 一把坚果 + 1个水煮蛋。撑住到晚餐别饿过头。
晚餐 20:00（最轻 · 21点前吃完）
一盘炒蔬菜 + 一掌鱼/豆腐/鸡 + 半碗主食。香肠少吃。吃完散步。

🍜吃粉 / 打卤面（不用戒）

加菜 + 加蛋白 + 面减半 + 先吃菜肉后吃面 + 饭后散步。
能选就选荞麦/全麦面、卤别勾太稠；想吃尽量放午餐，晚餐留给清淡的。

🍳蛋白来源（照顾胆固醇+尿酸）

✅ 多用

  • 无糖希腊酸奶 / 低脂奶（还降尿酸）
  • 蛋清（无胆固醇）
  • 豆腐 / 豆制品
  • 去皮鸡肉、适量鱼
  • 1 个全蛋 / 天

⚠️ 少吃

  • 虾、贝类、沙丁鱼（高嘌呤）
  • 红肉、动物内脏
  • 香肠等加工肉
  • 含糖饮料、果汁、酒
  • 奶酪别多（饱和脂肪）

🫐水果

✅ 多选

  • 蓝莓 / 草莓 / 树莓
  • 苹果、梨、猕猴桃
  • 柚子、橙子（适量）

⚠️ 少碰

  • 果干、果汁（浓缩果糖）
  • 荔枝、桂圆、芒果、榴莲
  • 葡萄、西瓜（很甜的）
吃整果不榨汁、配酸奶/坚果、每天 1–2 拳头。

🥜坚果 / 种子

每天一小把、选原味、换着吃（瓜子/核桃/杏仁轮流），全部加起来一把就够。
亚麻籽一定要磨碎——整粒吞下去吸收不了，等于白吃。

✋不用称重·用手估

🖐️
蛋白质（蛋/肉/鱼/豆腐）≈ 一个手掌大
✊
蔬菜 ≈ 两个拳头（多多益善）
👊
主食（饭/面/燕麦）≈ 一个拳头/一捧
👍
坚果/油 ≈ 一个拇指/一小把
先菜肉、后主食；每餐够一掌蛋白；
白天吃足、晚上吃轻，饭后散步。
不节食、不掉秤——稳血糖、好好吃就是养生。
以上为一般饮食建议，非个体化医疗/营养方案。
血脂、尿酸异常请定期复查，必要时遵医嘱。
